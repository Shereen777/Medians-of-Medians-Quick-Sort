# Median of Medians Quick Sort Visualizer

An interactive web-based visualizer for **Quick Sort using the Median of Medians pivot selection strategy**.  
The project shows the array state, comparisons, swaps, recursion depth, pivot movement, and a live recursive tree for the Median of Medians process.

## Features

- Visualizes Quick Sort step by step.
- Uses Median of Medians for pivot selection.
- Shows a recursive group tree for pivot selection.
- Highlights active subarray, groups, medians, pivot, comparisons, swaps, and sorted elements.
- Displays operation status, pivot value, comparisons, swaps, recursion depth, and progress.
- Includes controls for:
  - New random array
  - Start / pause animation
  - Step-by-step execution
  - Reset
  - Speed adjustment
  - Array size adjustment
- No external libraries or installation required.
